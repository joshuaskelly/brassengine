Module raycaster

Module for raycaster renderer

Usage:

    raycaster = require('raycaster')

-- Tile palette for rendering the map
tiles = {
   [1] = assets.get_texture('stonewall.gif)
}

-- Camera info
position = vector2.new()
direction = vector2.new(0, 1)
fov = 90

-- Sprite
sprite_texture = assets.get_texture('sprite.gif')
sprite_position = vector2.new()

-- Map
walls = assets.get_texture('walls.gif')
map = raycaster.Map.new(walls.width, walls.height)
map.walls = walls.pixels

renderer = raycaster.Renderer:new()
renderer:feature('fog', 16)

-- Render loop
renderer:clear('all')
renderer:camera(position, direction, fov)
renderer:render(map, tiles)
renderer:render(sprite_texture, sprite_position)

Renderer:camera (position, direction, fov)
Set renderer's camera data.

Parameters:

  • position vector2.vector2 Camera position.
  • direction vector2.vector2 Camera forward vector.
  • fov number Camera field of view in degrees.
Renderer:feature (name, value)

Access renderer's features. If just the feature name is provided, the value of that feature will be returned. If a value is provided, the feature will be set to that value.

Features:

  • 'fogdistance' number Distance at which fog is fully dark.
  • 'shadetable' texture Texture to use to shade colors using distance. Setting to nil will render full bright.
  • 'drawwalls' boolean Should walls be drawn?
  • 'drawfloors' boolean Should floors be drawn?
  • 'drawceilings' boolean Should ceilings be drawn?
  • 'wallbrightness' number, number North/south facing wall brightness, east/west facing wall brightness.

Parameters:

  • name string Feature name.
  • value optional any Value to set feature to.
